Letöltés

 

A program telepíthető és telepítést nem igénylő verzióban is letölthető.
Letöltés előtt azonban mindenképpen olvasd el a programra vonatkozó licencet és a hardver/szoftver követelményeket!
 
A futtatáshoz 1.3-as vagy annál nagyobb verziószámú Java futtatókörnyezet szükséges!
 

Telepíthető verzió (Install)

Fájlnév Méret Jogállás Dátum
ant-mahjonggsetup.exe 11,5 Mbyte Ingyenesen használható 2005.07.22.

 

 

A program telepítése az ant-mahjonggsetup.exe állomány futtatásával indítható. A célkönyvtár alapértelmezésben a C lemezegység Program Files\Ant-Mahjongg v1.2 mappája, mely a telepítés során megváltoztatható. A sikeres installálás után a játék a Start menüből, és az Asztalon létrejött parancsikonról egyaránt indítható.
